O que são tags semânticas?

São tags HTML que possuem função estrutural para reforçar a semântica (significado) do código, facilitando a interpretação de navegadores, principalmente em termos de acessibilidade.

Atualmente o papel da HTML não é apenas estruturar documentos para a web, mas também descrever o significado do conteúdo presente nesses documentos por meio de tags semânticas. DevMedia

Vamos conhecer algumas delas!!

Tag header

A tag < header > define um cabeçalho para a página.

Tag main

A tag < main > define o conteúdo principal da página.

Tag section

A tag < section > define as sessões da página. Uma página pode ter várias sessões, mas não mais de um header.

Tag article

A tag < article > define os artigos da página. O artigo é um conteúdo isolado, muito usado em blogs e páginas de coteúdo. Por exemplo em um blog um post pode ser um artigo.

Tag aside

A tag < aside > define uma sessão da página que está localizada ao lado do conteúdo principal. Por exemplo um menu lateral.

Tag nav

A tag < nav > define um conteúdo de navegação. Por exemplo uma barra de menu.

Tag footer

A tag < footer > define o rodapé da página. Está localizada geralmente no fim da página.
